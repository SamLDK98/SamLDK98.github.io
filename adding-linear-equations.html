Linear Algebra

Adding linear equations

What exactly does it mean to add equations?

Equations are not values, so it doesn't seem coherent to say we can add them.

Likewise, if we think of them as lines or solution sets, it makes no sense to add or subtract. We are not going to get more line or less line or more solutions or less solutions.

What we are actually adding are the like-terms.

Some people might say, Sam. That is painfully pedantic. You're getting hung up on semantics.

To those people I say, It builds up! These little semantical inconsistencies -- they build up!

So, that's the first thing: we aren't adding equations, we are adding the like-terms of equations.

$$E_{1}:2x-3y=4$$

$$E_{2}:4x-y=3$$

$$E_{3}:(2x+4x)+(-3y-y)=4+3$$

$$E_{3}:6x-4y=7$$

Okay. But what does that even mean?

Let's take an example:

Coefficients determine how much relative value a variable has.

If our equations were:

$$E_{1}:x-y=0$$

$$E_{2}:50x-10y=0$$

We see a 1 to 1 ratio between x and y being added to a 1 to five ratio, and we might think that the sum must be an equation that defines a ratio somewhere in the middle of these two.

However, this is not true.

Look at this example.

These all describe the same line and the same 2 : 1 ratio between $x$ and $y$ values.

$$2x-y=0$$

$$20x-10y=0$$

$$200x-100y=0$$

These all describe the same line and the same 1 : 2 ratio between $x$ and $y$ values.

$$x-2y=0$$

$$10x-20y=0$$

$$100x-200y=0$$

Logically, if we have a ratio that says, for every x take 2 y, and another that says, for every y take 2 x, we can see that combining them should result in for every 1 x take 1 y.

And this is true if we add the like-terms of our equations in this example:

$$E_{1}:2x-y=0$$

$$E_{2}:x-2y=0$$

$$E_{1}+E_{2}:x-y=0$$

However, play around with the widget to see what happens when we add the scaled up or down versions of these equations.

Evidently, the absolute magnitudes of the coefficients influence the resulting ratio of the new equation.
