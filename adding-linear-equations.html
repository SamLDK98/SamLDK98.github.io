Linear Algebra

Adding linear equations

Create different linear equations, add them together, and see what kind of lines they produce.

Pointwise means: for each input point, evaluate both functions there, then add the resulting numbers.

The word is needed because functions are not numbers themselves.

They are rules that assign numbers to points.

So function addition cannot mean: add the whole functions all at once in the same way you add $2+3$.

Instead, the addition happens at every point individually.

Hence: pointwise.

Suppose:

$f(x,y)=x+y$

$g(x,y)=x-y$

Now pick a point, say: $(3,1)$.

Evaluate each function there:

$f(3,1)=3+1=4$

$g(3,1)=3-1=2$

Now add the outputs: $4+2=6$.

So the new function should output $6$ at the point $(3,1)$.

Now do this for every point in the plane.

That entire collection of summed outputs defines a new function.

That is what $(f+g)(x,y)$ means.

So:

$(f+g)(x,y)=f(x,y)+g(x,y)$

is really saying: At each point $(x,y)$, evaluate both functions and add the results.

The important thing is:

the inputs are not added; the functions themselves are not merged mysteriously; their outputs are added at corresponding points.

Hence: pointwise addition.

You can think of functions as landscapes over space.

At every coordinate: one landscape has some height, and another landscape has some height.

Pointwise addition means: add the heights at matching coordinates.

This creates a new landscape.

Then: the zero-height contour of that new landscape becomes the new line/equation.
