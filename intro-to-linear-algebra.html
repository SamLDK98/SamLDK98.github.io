Linear Algebra

Introduction without New Notation

Imagine:

$$\begin{cases} 4x - 2y = 2\\3x + 5y = 7 \end{cases}$$

Let's temporarily forget the specific constants and replace them with variables.

$$\begin{cases} 4x - 2y = c_{1}\\3x + 5y = c_{2} \end{cases}$$

Imagine all the possible values of $c_{1}$ and $c_{2}$ as two axes — call it "output space".

We take some input $(x,y) = (7,6)$ and want to see what values it produces for $c_{1}$ and $c_{2}$.

We could calculate the outputs row by row by substituting our input values into the equations directly.

But let's try it a different way.

Let's imagine a space of all the possible values for $x$ and $y$ -- call in 'input space'.

The distance from (0,0) to (1,0) is a unit step along the $x$-axis, and (0,0) to (0,1) is a unit step along the $y$-axis.

Any coordinate in input space can be thought of as a combination of these steps. These 'steps' are known as basis directions.

For example, (2, 3):

$$2(1,0)\ \text{and}\ 3(0,1)\to (2, 0)\ \text{and}\ (0, 3)\to (2, 3)$$

However, we know that our $x$ and $y$ values are weighted by the coefficients $4$ and $3$, and $-2$ and $5$, respectively.

So in output space, our $x$ basis vector becomes $(4,3)$ and our $y$ basis vector becomes $(-2,5)$.

This means that as $x$ increase by 1 we no longer move 'one step along the $x$ direction and zero steps along the $y$ direction', but according to the transformed basis directions.

Below we can see the impact of increasing or decreasing our input variables on both the input and output spaces.

Any coordinate in input space can then be "mapped" to output space by taking these new steps:

$$x(4,3)\ \text{and}\ y(-2,5)$$

For example, to see where a point in input space ends up after performing this transformation:

$$\begin{gathered}(7,6)\mapsto 7(4,3) + 6(-2,5) \\ = (28,21) + (-12,30) \\ = (16, 51)\end{gathered}$$

So we can imagine our input space "transforming", by imagining the coordinates $(0,1)$ and $(1,0)$ moving to $(4, 3)$ and $(-2, 5)$, respectively, and every other following suit.

Now, remember that our equations did actually specify a particular output:

$$\begin{cases} 4x - 2y = 2\\3x + 5y = 7 \end{cases}$$

We can perform a linear transformation to see where this $(2, 7)$ target vector ends up.

The question is then, "What input vector moves to this location following a transformation?"

To answer this, we reverse the transformation by moving the transformed basis directions back to their original positions.

As the space transforms back into the original coordinate system, the target vector moves with it, eventually revealing the input vector that generated it.

This recovered input vector can then be decomposed into scaled combinations of the original basis vectors to show the actual $x$ and $y$ values that solve the system.
