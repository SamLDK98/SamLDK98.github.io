Linear Algebra

Cancelling variables to reveal coordinates

Here are two facts that enable something cool.

Firstly, as you may have noticed, when we add two linear equations, the point of intersection between them is also the point of intersection with the new line.

This makes sense because, if a point satisfies $E_{1}$ and $E_{2}$, then it must also satisfy $E_{1}+E_{2}$.

The shared solution survives all linear combinations.

Nice.

Secondly, if we add like-terms such that we cancel a variable, the new line is parallel to the corresponding axis, and perpendicular to the other.

That means something significant.

Lines parallel to an axis reveal the coordinate of the the axis they are perpendicular to.

That means we could:

  • Create a line parallel to the $x$-axis.
  • Create a line parallel to the $y$-axis.
  • And know that the point at which they intersect is the same as our original two lines.
  • This point of intersection would be the solution to both original equations.

Very nice!

For example, consider:

$$E_{1}:2x+3y=13$$

$$E_{2}:4x-y=5$$

We want to eliminate $y$.

Multiply $E_{2}$ by $3$:

$$3E_{2}:12x-3y=15$$

Now add:

$$E_{1}+3E_{2}$$

$$(2x+3y)+(12x-3y)=13+15$$

giving:

$$14x=28$$

$$x=2$$

(This constructed the vertical line.)

Next we construct the horizontal line by eliminating $x$.

Multiply $E_{1}$ by $-2$:

$$-2E_{1}:-4x-6y=-26$$

Now add:

$$-2E_{1}+E_{2}$$

$$(-4x-6y)+(4x-y)=-26+5$$

giving:

$$-7y=-21$$

$$y=3$$

This constructs a horizontal line.

The two new axis-parallel lines:

$$x=2$$

$$y=3$$

still intersect at exactly the same point as the original equations.

So by strategically scaling and combining equations, we transform complicated slanted lines into lines that directly reveal the coordinates of the shared solution.
