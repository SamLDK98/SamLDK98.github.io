Collapsing a coordinate system onto a line

Imagine a 2D coordinate plane.

Each coordinate encodes information about two independent directions.

$(3,-4)$

This denotes a location three positive units in one direction and four negative units in another.

There is no operation implied.

However, what would it mean if we summed every coordinate?

Say we added these two values together.

For example:

  • $(3, -4) \to -1$
  • $(1, 1) \to 2$
  • $(5, 9) \to 14$

How could we interpret these summed values?

We have lost any information about how far our point is from the center of the axis.

For example,

  • $(2, 3) \to 5$
  • $(100, -95) \to 5$

The first is very close to the center, the other is not.

We have also lost information about whether we are on the right or left side of the vertical axis, or above or below the horizontal axis.

So what information is maintained within these summed values?

Firstly, notice that points producing the same sum are not random.

For example:

  • $(2, 3) \to 5$
  • $(4, 1) \to 5$
  • $(10, -5) \to 5$

These points all lie on the same diagonal line.

So, let's say we have the summed value 3.

We no longer know exactly where the original coordinate was.

Many different points can produce the same sum.

However, we do learn something about how all points with this summed value are related.

For example:

  • if we move one unit right,
  • we must move one unit down to maintain the same sum.

This movement can be represented by the direction:

$$(1, -1)$$

meaning:

  • $x$ increases by 1,
  • while $y$ decreases by 1.

These changes cancel each other out, so the total remains unchanged.

However, movement in a different direction changes the sum.

For example, if we move:

  • one unit right,
  • and one unit up,

then both coordinates increase together:

$$(1,1)$$

and the summed value increases.

So:

  • movement along one direction preserves the sum,
  • while movement along another direction changes it.

So, every coordinate along a diagonal line sums to the same value. In other words, all of these coordinates are equivalent under this operation.

We can therefore lump all these coordinates together under the same summed value and represent them on a line.
